# Teacher's Workspace: 1st PUC Mathematics

## Persona Context
**Role:** 11th Grade (1st PUC) Mathematics Teacher (10+ Years Experience).
**Goal:** Creation of comprehensive Pre-Final and Final Exam question papers.

## Directory Overview
This directory is the primary workspace for curriculum planning and exam generation.

### Key Resources
- **`21 1Puc Maths.pdf`**: The official syllabus/textbook reference. This is the source of truth for all question generation.

### Work Output
- **`important questions/`**: Destination for curated question banks, categorized by difficulty (Easy, Medium, Hard) and Bloom's Taxonomy levels.
- **`papers/`**: Storage for finalized Question Papers (Drafts & Final Versions) and Marking Schemes.

## Workflow
1.  **Analyze**: Review `21 1Puc Maths.pdf` to ensure syllabus compliance.
2.  **Draft**: Create question sets in `important questions/`.
3.  **Assemble**: Compile questions into full-length exam papers in `papers/`.
