# Chapter 3: Trigonometric Functions - Important Questions

## 1 Mark Questions
1. **Question:** Convert 40° 20' into radian measure.
   **Answer:** $40^\circ 20' = 40 \frac{1}{3}^\circ = \frac{121}{3}^\circ$. Radians = $\frac{121}{3} \times \frac{\pi}{180} = \frac{121\pi}{540}$.
2. **Question:** Find the value of sin(765°).
   **Answer:** $\sin(2 \times 360^\circ + 45^\circ) = \sin 45^\circ = \frac{1}{\sqrt{2}}$.
3. **Question:** Find the principal solution of tan x = √3.
   **Answer:** $\tan x = \tan(\pi/3)$. Solutions in $[0, 2\pi)$ are $\pi/3$ and $\pi + \pi/3 = 4\pi/3$.

## 2 Mark Questions
1. **Question:** Prove that: sin²(π/6) + cos²(π/3) - tan²(π/4) = -1/2.
   **Answer:** $(1/2)^2 + (1/2)^2 - (1)^2 = 1/4 + 1/4 - 1 = 1/2 - 1 = -1/2$.
2. **Question:** Find the value of tan(15°).
   **Answer:** $\tan(45-30) = \frac{\tan 45 - \tan 30}{1 + \tan 45 \tan 30} = \frac{1 - 1/\sqrt{3}}{1 + 1/\sqrt{3}} = \frac{\sqrt{3}-1}{\sqrt{3}+1} = 2 - \sqrt{3}$.
3. **Question:** If cos x = -3/5, x lies in the third quadrant, find the values of other five trigonometric functions.
   **Answer:** $x$ in Q3 $\implies \tan, \cot > 0$ and others $< 0$.
   $\cos x = -3/5 \implies \sin x = -4/5$.
   $\tan x = 4/3, \cot x = 3/4, \sec x = -5/3, \text{cosec } x = -5/4$.

## 3 Mark Questions
1. **Question:** Prove that: (cos 4x + cos 3x + cos 2x) / (sin 4x + sin 3x + sin 2x) = cot 3x.
   **Answer:** Use $C+D$ formulas. Group $(\cos 4x + \cos 2x) + \cos 3x$.
   Num: $2\cos 3x \cos x + \cos 3x = \cos 3x(2\cos x + 1)$.
   Den: $2\sin 3x \cos x + \sin 3x = \sin 3x(2\cos x + 1)$.
   Result = $\cot 3x$.
2. **Question:** Prove that: tan 4x = (4 tan x (1 - tan² x)) / (1 - 6 tan² x + tan⁴ x).
   **Answer:** Use $\tan 2(2x) = \frac{2 \tan 2x}{1 - \tan^2 2x}$. Substitute $\tan 2x = \frac{2\tan x}{1-\tan^2 x}$ and simplify.

## 5 Mark Questions
1. **Question:** Prove that: cos² x + cos²(x + π/3) + cos²(x - π/3) = 3/2.
   **Answer:** Expand using $2\cos^2 \theta = 1 + \cos 2\theta$.
   $LHS = \frac{1}{2} [ (1+\cos 2x) + (1+\cos(2x+2\pi/3)) + (1+\cos(2x-2\pi/3)) ]$
   $= \frac{1}{2} [ 3 + \cos 2x + 2\cos 2x \cos(2\pi/3) ]$
   $= \frac{1}{2} [ 3 + \cos 2x + 2\cos 2x (-1/2) ]$
   $= \frac{1}{2} [ 3 + \cos 2x - \cos 2x ] = 3/2$.
2. **Question:** Prove that: sin x + sin 3x + sin 5x + sin 7x = 4 cos x cos 2x sin 4x.
   **Answer:** Group $(\sin 7x + \sin x) + (\sin 5x + \sin 3x)$.
   $= 2\sin 4x \cos 3x + 2\sin 4x \cos x$
   $= 2\sin 4x (\cos 3x + \cos x)$
   $= 2\sin 4x (2\cos 2x \cos x) = 4\cos x \cos 2x \sin 4x$.
