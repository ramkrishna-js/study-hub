# Chapter 2: Relations & Functions - Important Questions

## 1 Mark Questions
1. **Question:** If (x + 1, y - 2) = (3, 1), find the values of x and y.
   **Answer:** $x+1=3 \implies x=2$. $y-2=1 \implies y=3$.
2. **Question:** If A = {1, 2}, find A × A × A.
   **Answer:** $\{(1,1,1), (1,1,2), (1,2,1), (1,2,2), (2,1,1), (2,1,2), (2,2,1), (2,2,2)\}$.
3. **Question:** Define a Signal Function and write its range.
   **Answer:** $f(x) = |x|/x$ for $x \neq 0$, $0$ for $x=0$. Range = $\{-1, 0, 1\}$. (Note: User likely meant "Signum" function).

## 2 Mark Questions
1. **Question:** Find the domain and range of the real function f(x) = √(9 - x²).
   **Answer:** Domain: $9-x^2 \ge 0 \implies x^2 \le 9 \implies -3 \le x \le 3$. Range: $[0, 3]$.
2. **Question:** If f(x) = x² and g(x) = 2x + 1, find (f + g)(x) and (f - g)(x).
   **Answer:** $(f+g)(x) = x^2 + 2x + 1$. $(f-g)(x) = x^2 - 2x - 1$.

## 3 Mark Questions
1. **Question:** Let A = {1, 2, 3, 4, 6}. Let R be the relation on A defined by {(a, b) : a, b ∈ A, b is exactly divisible by a}.
   (i) Write R in roster form.
   (ii) Find the domain and range of R.
   **Answer:** (i) $R = \{(1,1), (1,2), (1,3), (1,4), (1,6), (2,2), (2,4), (2,6), (3,3), (3,6), (4,4), (6,6)\}$.
   (ii) Domain = $\{1, 2, 3, 4, 6\} = A$. Range = $\{1, 2, 3, 4, 6\} = A$.
2. **Question:** Define the Modulus function. Draw its graph and write its domain and range.
   **Answer:** $f(x) = |x|$. Domain = $R$. Range = $[0, \infty)$. Graph is V-shaped at origin.

## 5 Mark Questions
1. **Question:** Define the Greatest Integer Function. Draw its graph and write its domain and range.
   **Answer:** $f(x) = [x]$ (greatest integer $\le x$). Domain = $R$. Range = $Z$ (Integers). Graph is a step function.
2. **Question:** Let f = {(1,1), (2,3), (0,-1), (-1,-3)} be a linear function from Z to Z. Find f(x).
   **Answer:** Let $f(x) = mx + c$.
   $f(0) = -1 \implies m(0) + c = -1 \implies c = -1$.
   $f(1) = 1 \implies m(1) - 1 = 1 \implies m = 2$.
   $f(x) = 2x - 1$.
